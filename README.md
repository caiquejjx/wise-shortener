## Descrição
Encurtador de URLs

## Iniciar a aplicação

```bash
$ docker-compose up
```

## Testes

```bash
# start container
$ docker-compose up

#unit tests
$ docker exec -it wise-shortener sh -c "yarn test"

#e2e
$ docker exec -it wise-shortener sh -c "yarn test:e2e"

#coverage
$ docker exec -it wise-shortener sh -c "yarn test:cov"

```
